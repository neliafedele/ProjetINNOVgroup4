# ProjetINNOVgroup4

Checklist de test ultra rapide :

- Ouvre l’app dans le navigateur via ton serveur local (pas en double-clic fichier).
- Vérifie l’installation PWA : menu navigateur → “Installer l’application”.
- Connecte-toi en mode étudiant, sélectionne 2–3 jours, puis clique “Confirmer ma présence”.
- Recharge la page puis passe en mode personnel : les KPI “midi/soir” doivent refléter les votes oui.
- Coupe internet et recharge : la page doit encore s’ouvrir (cache service worker).
- Si la version ne se met pas à jour, vide les données du site/service worker puis recharge une fois.
